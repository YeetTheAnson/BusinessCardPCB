## Features
 - Programmable 13.56MHz NFC and QR Code to redirect to Github page
 - 8*8 Neopixel RGB Matrix

## Parts
- Adafruit NeoPixel 4020 Side-Light RGB LED
- NTP53321G0JTZ RFID IC 13.56MHz NFC
- CH340G USB serial converter
- ESP-12E
